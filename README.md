## 🚀 Features

- 📰 Fetch latest news using NewsAPI
- 📂 Multiple categories (Technology, World, AI)
- 🤖 AI-generated summaries
- 💾 Save summaries to file (`summaries.json`)
- 🖥️ Simple CLI interface

---

## 🛠️ Tech Stack

- Python  
- Requests  
- Transformers (Hugging Face)  
- JSON  
- NewsAPI

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Vishwam-Gawande/ai-news-assistant.git
cd ai-news-assistant

2️⃣ Install dependencies

pip install requests transformers torch

3️⃣ Add your API key

Open main.py and replace:

API_KEY = "YOUR_API_KEY"

▶️ Usage

python main.py
Steps:

Choose a category

Select a news article

View AI-generated summary

📁 Output

Summaries are saved in:

summaries.json

🧠 How It Works

1. Fetches news from API

2. Displays top headlines

3. Takes user input

4. Uses AI to summarize

5. Saves output locally
